一个跨设备同步绘制的白板库。
类型支持曲线、直线、圆形、矩形、文字、橡皮擦，
样式支持设置颜色、粗细、透明度，支持设置图片作为背景，
操作支持undo、redo、保存png图片，获取绘制数据&从绘制数据恢复画板。
